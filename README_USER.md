🧠 How to Use the Quiz Website
Welcome to the quiz! This site is designed to be quick, intuitive, and fun. Follow the steps below to get the most out of it.

🚀 Getting Started
Open the quiz homepage.

Click Start Quiz to begin.

Each question will appear one at a time.

📝 Answering Questions
1. Multiple‑Choice Questions
Read the question carefully.

Select one of the available answer options.

Your selection will be highlighted.

Click Next to continue.

2. User Input Questions
Some questions require you to type your answer.

Click the input box and enter your response.

When you're ready, press Next.

🔄 Navigation
Use the Next button to move forward.

If enabled, the Back button lets you revisit previous questions.

Your answers are saved automatically as you progress.

🏁 Finishing the Quiz
After the final question, you’ll see a Results page.

This page may show:

Your score

Correct answers

Feedback or explanations (if included)
